# ✂️ TrimBG | AI Background Remover
### Professional AI-Powered Background Removal - 100% Private, 100% Local.
### الأداة العربية لإزالة الخلفية بالذكاء الاصطناعي - خصوصية تامة ومعالجة محلية.

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge&logo=netlify)](https://tr1mbg.netlify.app)
[![GitHub License](https://img.shields.io/github/license/mohamed-khairy-5i/TrimBG?style=for-the-badge)](LICENSE)

**TrimBG** is a cutting-edge web application designed to remove image backgrounds with professional precision directly in your browser. Unlike other tools, **TrimBG** processes everything locally using your device's hardware, ensuring your images never leave your computer.

---

## ✨ Key Features | المميزات الرئيسية
- **100% Local Processing**: No images are ever uploaded to a server. Privacy is built-in by design.
- **معالجة محلية 100%**: لا يتم رفع صورك إلى أي خادم، الخصوصية هي الأولوية.
- **Zero Data Retention**: We don't collect, store, or see your data.
- **عدم الاحتفاظ بالبيانات**: لا نقوم بجمع أو تخزين أو الاطلاع على بياناتك.
- **Premium UI/UX**: A sleek, modern, and responsive interface designed for the best user experience.
- **تجربة مستخدم مميزة**: واجهة عصرية وسريعة الاستجابة مصممة لأفضل تجربة.
- **High Precision**: Powered by advanced AI models (WASM/ONNX).
- **دقة عالية**: مدعوم بنماذج ذكاء اصطناعي متطورة تعمل داخل المتصفح.

## 🛠️ Tech Stack | التقنيات المستخدمة
- **Core**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **AI Engine**: WebAssembly (WASM) & ONNX Runtime Web
- **Components**: Shadcn UI & Lucide React

---

## 🚀 Getting Started | كيفية البدء

### Prerequisites | المتطلبات
- Node.js installed on your system.

### Installation | التثبيت

1. **Clone the repository | تحميل المشروع:**
   ```bash
   git clone https://github.com/mohamed-khairy-5i/TrimBG.git
   cd TrimBG
   ```

2. **Install dependencies | تثبيت المكتبات:**
   ```bash
   npm install
   ```

3. **Run Locally | التشغيل المحلي:**
   ```bash
   npm run dev
   ```
